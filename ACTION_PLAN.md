### 3. Manual Testing ‚è±Ô∏è 1 hour

**Test Authentication:**
```bash
# Register user
curl -X POST http://localhost:8000/auth/register \
  -H "Content-Type: application/json" \
  -d '{"name":"Test User","email":"test@example.com","password":"test123","org_name":"Test Org"}'

# Login
curl -X POST http://localhost:8000/auth/login \
  -H "Content-Type: application/json" \
  -d '{"email":"test@example.com","password":"test123"}'
```

**Test Persona Creation:**
```bash
# Create persona (use token from login)
curl -X POST http://localhost:8000/personas \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"name":"Math Tutor","description":"A helpful math teacher"}'
```

**Test Knowledge Upload:**
```bash
# Upload a PDF
curl -X POST http://localhost:8000/knowledge/upload \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -F "file=@sample.pdf" \
  -F "persona_id=PERSONA_ID" \
  -F "title=Sample Document"
```

**Checklist:**
- [ ] User registration works
